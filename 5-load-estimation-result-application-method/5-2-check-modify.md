# 5.2 부하추정 결과 확인 및 수정

추정된 부하의 정보는 『**설정**』 → 『**3.로봇파라미터**』 → 『**1.툴데이터**』 메뉴에서 확인 및 수정할 수 있습니다. 추정할 때 입력한 툴번호에서 추정된 결과가 나타납니다.

![그림 11 툴 데이터 화면](<../.gitbook/assets/image (7).png>)

{% hint style="info" %}
이너셔의 경우, 추정 결과는 부하의 질량 중심에 대한 이너셔입니다.\
특정방향에 대한 이너셔가 작은 경우에는, 추정결과가 0으로 표시될 수 있습니다.
{% endhint %}
