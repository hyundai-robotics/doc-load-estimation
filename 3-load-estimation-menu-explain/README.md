# 3. 부하추정 메뉴 설명

『**설정**』 → 『**6.자동 캘리브레이션**』 → 『**4.부하추정 기능**』에서 부하추정을 실행합니다.

{% hint style="info" %}
『**4.부하추정 기능**』 메뉴를 선택할 때, 제어 모드가 ‘**제진제어**’이면 ‘**PPI**’ 모드로 변경하기 위해 자동으로 모터가 ‘**Off**’ 됩니다. 부하추정 종료시 다시 ‘**제진제어**’모드로 변경되며 모터 ‘**Off**’됩니다.
{% endhint %}
